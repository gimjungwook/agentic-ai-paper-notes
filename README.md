# Agentic AI 정독 — Fall 2027 교수 컨택 준비

Fall 2027 미국 PhD 지원에서 컨택할 1순위 4인 교수의 핵심 논문 8편을 arXiv 원문 기반으로 정독한 인터랙티브 사이트입니다. 각 논문을 *문제와 동기 → 핵심 아이디어 → 결과 → 한계 → 내 연구(장기기억 Knowledge Base)와의 연결 → 컨택 이메일 문장* 순서로 풀고, 핵심 메커니즘마다 직접 만져볼 수 있는 위젯을 붙였습니다.

> **라이브 사이트**: <https://gimjungwook.github.io/agentic-ai-paper-notes/>

## 정리

- 김정욱 (Jungwook Kim) · 연구 주제: AI 에이전트를 위한 장기기억 기반 Knowledge Base
